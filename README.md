# Firebase Authentication Assignment

## 📌 Project Overview

This project is a Firebase Authentication System built using HTML, CSS, JavaScript, and Firebase Authentication.

The application allows users to:

* Create a new account (Signup)
* Login with email and password
* Access a protected Home Page
* View their email address after login
* Logout securely
* Stay logged in until logout
